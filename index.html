Minhas Skills

HTML

HTML (HyperText Markup Language) é uma linguagem base usada para criar e estruturar páginas da web. Ele organiza o conteúdo em uma estrutura que os navegadores interpretam e exibem para os usuários.

CSS

CSS (Cascading Style Sheets) é a linguagem usada para estilizar páginas da web. Ela complementa o HTML, que organiza o conteúdo, ao definir como esse conteúdo deve ser visualmente apresentado.

JAVA SCRIPT

JavaScript (JS) é uma linguagem de programação que permite adicionar interatividade e funcionalidade às páginas web.

Minha Escolaridade
